# Tests

| Directory | Description                                                                                                                 |
| :-------- | :-------------------------------------------------------------------------------------------------------------------------- |
| `bench`   | Benchmarks.                                                                                                                 |
| `integ`   | Integration tests including some tests of units which cannot be tested in unit tests due to death tests or file operations. |
| `units`   | Unit tests. Time-consuming tests should be avoided as much as possible because unit tests are run frequently.               |
| `utils`   | Utilities for tests.                                                                                                        |
